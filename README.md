# MusicPlayer_In_Android

Music Player in Android using JAVA.

Pre-requisites
--------------

- Android SDK v28
- Min. Android SDK v21

Features
---------------
- Songs External Music Storage
- Play/Pause Song
- Adaptive Layout
- Search Song in local Storage
- Open Source


Extra Dependencies
------------------
````
implementation 'com.karumi:dexter:5.0.0'
````
